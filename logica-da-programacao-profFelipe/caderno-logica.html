Função

Função com parametro

Para chamar a função você escreve assim:

torrar (pão de forma)

O que tá dentro do () acima, é um parâmetro, entenda ele como os ingredientes para fazer o bolo, enquanto a função é o bolo


       function torrar (pão) {
        console.log ("torrada feita com " + pao)
       }

Dessa forma o console será:
torrada feita com pão de forma

Escopo de variáveis

Se você chamar um console, fora do escopo "`{}"


        console.log(pao)

Dá erro. Ela só existe dentro do contexto - dentro das chaves

Cuidados com os parametros

Se você não passar um valor em JS, ele aparece undefined. Se ninguem formar o nome no seu pedido, você pode mudar esse nome.
